Remanso del Marinero
 
Sitio web en desarrollado con el objetivo de brindar información clara y accesible sobre Remanso del Marinero. A través de un diseño simple e intuitivo, los visitantes pueden conocer sus principales características, servicios, galería de imágenes y datos de contacto.

Este proyecto fue creado priorizando una experiencia de navegación cercana, amigable y adaptable a distintos dispositivos

Tecnologías utilizadas:

HTML5
CSS3
JavaScript
Bootstrap
